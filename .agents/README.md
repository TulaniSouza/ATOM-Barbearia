# Time de Agentes de IA - ATOM Barbearia

Este diretório contém as definições e instruções para os agentes de IA especializados que compõem o time de desenvolvimento do projeto ATOM Barbearia. Cada agente é otimizado para a stack tecnológica do projeto (React, SCSS, Vite) e possui ferramentas específicas para sua função.

## Índice de Agentes

| Agente | Descrição | Quando Usar |
| :--- | :--- | :--- |
| **[System Strategist](./system-strategist.md)** | O cérebro do time. Analisa problemas complexos, cria planos de ação e delega tarefas. | No início de qualquer tarefa complexa ou nova funcionalidade. |
| **[Frontend Coder](./frontend-coder.md)** | Especialista em React e SCSS. Escreve código moderno e performático seguindo o design system. | Para implementar componentes, telas e estilos baseados no design. |
| **[QA & E2E Tester](./e2e-tester.md)** | Especialista em testes automatizados com Playwright. Valida funcionalidades e interface. | Após implementações para garantir que tudo funciona e o design está correto. |

## Fluxo de Trabalho Recomendado

1.  **Análise:** O `System Strategist` analisa a solicitação e define o plano.
2.  **Execução:** O `Frontend Coder` implementa as mudanças solicitadas.
3.  **Validação:** O `QA & E2E Tester` verifica a implementação no navegador.
