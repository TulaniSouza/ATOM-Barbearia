## Sobre o Projeto
O **Hora Marcada Barbearia** é uma solução de gestão de agendamentos focada em barbeiros autônomos. O projeto nasceu da necessidade de resolver a sobrecarga administrativa de profissionais como o **Vicente Cavalcante** (nossa Persona), que perdem tempo produtivo organizando horários manualmente via WhatsApp.

Este projeto faz parte do desafio **Labs Talents 2026**.

## Tecnologias e Ferramentas
- **Automação:** n8n + AI Agents (LLM)
- **Banco de Dados:** PostGree / Modelagem ER
- **Design:** Figma (UX/UI Lean Inception)
- **Gestão:** Trello (Metodologia Ágil)

## Solução (MVP)
Nosso foco é um "MVP Enxuto" que resolve o problema central:
- **Visualização da Agenda:** Tela única para controle diário.
- **Agendamento Inteligente:** Fluxo automatizado para redução de conflitos de horário.
- **Onboarding do Barbeiro:** Facilidade na configuração inicial da disponibilidade.

## Segurança & Governança (Blue Team Mindset)
- **LGPD:** Aplicação de princípios de minimização de dados.
- **Segurança de APIs:** Implementação de boas práticas de autenticação e consumo de APIs REST.
- **Integridade:** Normalização de banco de dados (1FN-3FN) para evitar redundância e erros de agendamento.
